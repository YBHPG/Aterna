# Aterna

## Описание

Бэкенд-часть веб-приложения Aterna. Проект построен на базе современной Node.js архитектуры с использованием фреймворка NestJS, обеспечивающего модульность, масштабируемость и строгую типизацию.

## Используемый технологический стек

- **Платформа:** Node.js
- **Фреймворк:** [NestJS](https://nestjs.com/)
- **Язык программирования:** TypeScript
- **Базы данных:**
    - PostgreSQL (ORM: [TypeORM](https://typeorm.io/)) — для реляционных данных (пользователи, авторизация).
    - MongoDB (ODM: [Mongoose](https://mongoosejs.com/)) — для документ-ориентированных данных.
- **Безопасность и криптография:** `bcrypt` (хэширование паролей)
- **Валидация:** `joi`
- **Тестирование:** Jest

## Проделанная работа

### 1. Базовая инициализация проекта

- Развернут каркас приложения на NestJS.
- Настроена конфигурация TypeScript (`tsconfig.json`) и игнорируемые файлы (`.gitignore`).
- Установлены все ключевые зависимости (БД, криптография, тестирование).
- Настроена среда для выполнения изолированных Unit-тестов с помощью Jest.

### 2. Проектирование реляционной схемы данных пользователя

Реализован базовый модуль `Users` для управления аутентификационными данными пользователей в PostgreSQL:

- **Entity (`User`):** Создана сущность пользователя с полями: уникальный идентификатор (`UUID`), нормализованный `email` (уникальный индекс), криптографический хэш пароля (`passwordHash`) и временные метки (`createdAt`, `updatedAt`).
- **Service (`UsersService`):**
    - Метод `findByEmail`: Поиск пользователя по email.
    - Метод `create`: Создание нового пользователя. Включает бизнес-логику приведения почты к нижнему регистру и криптографического хэширования пароля через `bcrypt` (salt rounds = 10).
- **Тестирование:** Написан Unit-тест (`users.service.spec.ts`), изолированно проверяющий логику нормализации email и вызова функции хэширования с мокированием репозитория базы данных.

## Разработка и запуск

### Установка зависимостей

```bash
npm install
```

### Запуск приложения

```bash
# Режим разработки с hot-reload
npm run start:dev
```

### Запуск тестов

```bash
npm run test
```
